# OSS-Audit-24BAI10128

This repository contains a set of Shell scripts (`script1.sh` through `script5.sh`) designed for execution in a Linux/Ubuntu environment.

---

##  Prerequisites

Before running the scripts, ensure your Ubuntu system is prepared.

1. **Update Package List**:
   ```bash
   sudo apt update
2. **Verify Bash Shell**:
Check if Bash is available on your system:
```bash
   bash --version
```

##  Installation & Setup
1. **Clone the Repository**:

Open your Ubuntu terminal and run the following commands:
```bash

git clone [https://github.com/ShadowCodster/OSS-Audit-24BAI10128.git](https://github.com/ShadowCodster/OSS-Audit-24BAI10128.git)
cd OSS-Audit-24BAI10128
```
2. **Grant Executable Permissions**:

You must give the system permission to run these scripts. Run this command to authorize all scripts at once:
```bash

chmod +x *.sh
```
## Running the Scripts

To execute a script, use the ./ prefix followed by the script name in your terminal:
Script Name	Command to Execute
Script 1	./script1.sh
Script 2	./script2.sh
Script 3	./script3.sh
Script 4	./script4.sh
Script 5	./script5.sh
